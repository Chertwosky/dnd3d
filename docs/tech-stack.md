# Tech Stack

Документ фиксирует базовый технологический стек проекта DnD 3D.

## Frontend

- **Next.js** — основной фреймворк для frontend-приложения.
- **React** — библиотека для построения пользовательского интерфейса.
- **TypeScript** — основной язык разработки frontend-кода.
- **Tailwind CSS** — утилитарный CSS-фреймворк для стилизации интерфейса.
- **Zustand** — библиотека для управления клиентским состоянием.
- **React Hook Form** — библиотека для работы с формами.
- **Zod** — библиотека для валидации схем и типизации данных.

## Backend

- **Node.js** — runtime для backend-сервисов.
- **NestJS** — основной backend-фреймворк.
- **PostgreSQL** — основная реляционная база данных.
- **Prisma ORM** — ORM для работы с базой данных и миграциями.
- **Redis** — хранилище для кеширования, сессий и realtime-сценариев.
- **WebSocket** — транспорт для realtime-взаимодействия.

## Visual Modules

- **Three.js / React Three Fiber** — 3D-графика и рендеринг 3D-аватара.
- **Konva.js** — библиотека для интерактивной 2D-карты.
